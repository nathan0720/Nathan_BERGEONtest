Spirale en carré

Explication

Boucles imbriquées : une boucle while pour la progression de la spirale et une boucle for pour dessiner le sous-motif (un petit triangle) à chaque étape. Maîtrise de la rotation à 90° pour le tracé carré.

Simulateur Interactif

Lancer le simulateur avec le bouton run

Le Code Python

from turtle import *

x = 10

while x < 150:
    for i in range(3):
         fd(x/5)
         left(120)
    fd(x)
    left(90)
    x += 10

done()
