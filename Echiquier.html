Echiquier

Explication

Application de boucles for imbriquées pour itérer sur les rangées et les colonnes du plateau. Utilisation d'une structure conditionnelle (if/else) et de l'opérateur de parité (% 2) pour alterner la couleur de chaque case. Dessin de formes pleines (begin_fill, end_fill).

Simulateur Interactif

Lancer le simulateur avec le bouton run

Le Code Python

from turtle import *
#centre l'échiquier
COTE_CASE = 20
POSITION_DEPART = -80
ANGLE_DROIT = 90
NB_CASES = 8

# Configuration initiale
speed(0)  # Accélère le tracé
penup()
goto(POSITION_DEPART, POSITION_DEPART)
pendown()

for k in range(NB_CASES): # Boucle pour les colonnes (axe Y)
    for j in range(NB_CASES): # Boucle pour les rangées (axe X)
        
        # Définition de la couleur selon la parité de la case
        if (k + j) % 2 == 0:
            fillcolor("black")
        else:
            fillcolor("white")
            
        begin_fill()
        # Dessin d'une case carrée
        for i in range(4):
            forward(COTE_CASE)
            left(ANGLE_DROIT)
        end_fill()
        
        # Avancer pour la prochaine case de la rangée
        forward(COTE_CASE)
        
    # Retour au début de la ligne et passage à la ligne supérieure
    penup()
    goto(POSITION_DEPART, ycor() + COTE_CASE)
    pendown()

done()
