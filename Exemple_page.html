Python Turtle

PROGRAMMATION 2024

À propos du projet

Ce projet a été réalisé dans le cadre de l'apprentissage du **Python** au lycée, en utilisant la librairie **Turtle** pour l'initiation à la programmation graphique. L'objectif était de créer des formes géométriques et des dessins complexes en exploitant les boucles et les fonctions, permettant ainsi de visualiser immédiatement l'impact du code.

Démonstration et Code Turtle

Voici une démonstration interactive d'une de mes créations utilisant la librairie Turtle, affichant le dessin généré ainsi que le code Python associé.

Code Python (Exemple d'un dessin simple) 

from turtle import *
import colorsys

speed(0)
hideturtle()
bgcolor('black')
pensize(2)

def draw_design(n):
    h = colorsys.rgb_to_hsv(0.0, 0.8, 0.8)[0]
    
    for i in range(150):
        c = colorsys.hsv_to_rgb(h, 1, 1)
        pencolor(c)
        
        circle(100 - i * 0.5, 90)
        left(90)
        circle(100 - i * 0.5, 90)
        right(180)
        
        h += 0.005 # Change la couleur progressivement

draw_design(150)
done()

Défis Techniques

Objectifs:

Développement de figures récursives.

Création de fractales avec la tortue.

Utilisation des boucles `for` pour des motifs répétitifs.

Gestion des couleurs et de la taille du stylo pour un rendu esthétique.
