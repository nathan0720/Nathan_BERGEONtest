Oeuf

Explication

Application de concepts mathématiques et géométriques avancés, notamment l'utilisation du Théorème de Pythagore pour calculer le rayon précis de l'arc supérieur. Utilisation de la fonction write() pour étiqueter les points de construction.

Simulateur Interactif

Lancer le simulateur avec le bouton run

Le Code Python

from turtle import *

"""
mon simulateur python ne prend pas 
en charge "numinput" donc j'ai code
le rayon en dure.
r = int(numinput("Choisis", "Taille 
de l'oeuf (moyen 100)?")) 
"""
r = 100 #valeur que l'on veut

#pour bien se place au centre
penup()
goto(-r, -r/2)
pendown()


right(90)
write('A')
circle(r, 180)
write('B')
circle(r*2,45)
write('E')
# on utilise le théorème de Pythagore
circle(r*2-(r**2+r**2)**0.5,90)
write('D')
circle(r*2,45)
write('A')

done()
